feat: implement notification gatekeeper and registry toggle

This commit introduces the Notification Gatekeeper C# application 
which intercepts and batches Windows notifications using the 
UserNotificationListener API. It also includes the necessary 
AppxManifest.xml for local registration and a PowerShell toggle 
script to easily enable or disable the background application and 
the corresponding Windows Registry settings for targeted apps (#1).
